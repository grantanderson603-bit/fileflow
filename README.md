# FileFlow — Smart File Organizer

AI-powered file renaming, organization, and deduplication.

## Features
- Scan and analyze files with AI vision
- - Smart rename based on content
  - - Find and remove duplicates
    - - Auto-organize into folders
     
      - ## Setup
      - ```bash
        pip install -r requirements.txt
        python app.py
        ```

        Visit http://localhost:5050
